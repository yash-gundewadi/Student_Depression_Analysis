## Problem Statement

This project analyzes student depression data to understand mental health patterns and identify key factors affecting students.

The goal is to:
- Identify major causes of student depression  
- Analyze trends across demographics (age, gender, etc.)  
- Study the impact of academic pressure and lifestyle  
- Provide insights for improving student well-being  

---

## Steps followed

- Step 1 : Load raw dataset into SQL  
- Step 2 : Clean data (handle nulls, remove duplicates, fix data types)  
- Step 3 : Transform and prepare dataset for analysis  
- Step 4 : Export cleaned dataset  
- Step 5 : Import dataset into Tableau  
- Step 6 : Created calculated fields   
- Step 7 : Build interactive dashboard  
- Step 8 : Add filters, visuals, and finalize design  

---

## Key Analysis

### Data Cleaning (SQL)
- Removed missing and inconsistent values  
- Standardized categorical data  
- Ensured proper data types  

### Visualization (Tableau)
- Depression distribution among students  
- Academic pressure vs depression levels  
- Study hours and lifestyle impact  
- Demographic analysis (age, gender)  

---

## Insights

### Overall Trends
- Academic pressure is a major factor in depression  
- Certain student groups show higher depression levels  

### Behavioral Insights
- Lifestyle habits and study patterns influence mental health  
- Higher stress correlates with increased depression levels  

### Demographic Insights
- Noticeable variation across gender and age groups  
- Some segments require targeted support  

---

## Conclusion

This project provides valuable insights into student mental health using SQL and Tableau. It helps in identifying key stress factors and supports data-driven decisions to improve student well-being.
